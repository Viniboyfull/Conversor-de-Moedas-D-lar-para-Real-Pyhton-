Conversor de Moedas de Dólar para Real.

Instruções:

1- Copiar o Código para o VS Code ou Compilador Pyhton.

2 - Caso o interptetador não reconheça instale o módulo requests no seu ambiente, para isso abra o  Powershell e digite o comando "pip install requests".

OBS: você pode transformar em um executável usando o Pyinstaller, usando o comando "pyinstaller --onefile --windowed Conversor_de_moedas.py"
